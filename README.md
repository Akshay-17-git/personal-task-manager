## ğŸ“– Usage

1. **Adding Tasks:**
   - Enter task description in the input field
   - Select priority level from dropdown
   - Optionally set a due date
   - Click "Add Task" or press Enter

2. **Managing Tasks:**
   - Click "Done" to mark tasks complete
   - Click "Edit" to modify task text
   - Click "Delete" to remove tasks (with confirmation)
   - Use filter buttons to view All, Pending, or Done tasks
   - Use search bar to find specific tasks

3. **Data Persistence:**
   - All tasks are automatically saved to browser's local storage
   - Data persists between browser sessions
